## Install & Run (3 Steps)

### Step 1: Install Dependencies
```bash
cd ASSIGNMENT_1
npm install
```

### Step 2: Add Your API Key
Open `src/App.jsx` and replace line 8:
```javascript
const API_KEY = 'YOUR_API_KEY_HERE'
```

Get free API key from: https://newsapi.org/

### Step 3: Run the App
```bash
npm run dev
```

Open browser to: `http://localhost:5173`
